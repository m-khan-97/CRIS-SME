# CRIS-SME

Cloud Risk Intelligence System for Small & Medium Enterprises

## Overview
CRIS-SME is an AI-assisted cloud governance and compliance risk assessment framework designed to help SMEs identify, score, and manage security and governance risks in Azure cloud environments.

## Key Features
- Azure governance control checks
- Risk scoring and prioritisation
- Compliance mapping
- Explainable reporting
- SME-focused design

## Research Motivation
Many SMEs lack accessible, automated tools for cloud governance and compliance risk assessment. CRIS-SME addresses this gap through a lightweight, explainable, and extensible framework.
